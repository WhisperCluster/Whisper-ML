## Setup
To run this project,
   Download Cluster WHISPER Data products from CSA, https://csa.esac.esa.int/csa-web/#search
   Install CEFLIB, https://ceflib.irap.omp.eu/ 
   Install python and python environement requieremnts
   Run Demo Files
